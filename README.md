# unbex-web
Unbex official website — Frontend development

## Formulario de postulaciones (Trabajá con nosotros)

El formulario usa [Formspree](https://formspree.io) para enviar los mails sin backend.  
Endpoint actual: `https://formspree.io/f/meepnevk` (configurado en la cuenta de Fran).

**Antes de pasar a producción:**
- Reemplazar el endpoint en `trabajar-con-nosotros.html` por el de la cuenta definitiva de Unbex si se crea una nueva.

**Primera vez que se usa:**
Formspree manda un mail de confirmación al correo configurado en la cuenta. Hay que hacer click en ese link una sola vez para activar el endpoint. Recién ahí empiezan a llegar las postulaciones normalmente.
